Software

Loopassembly.info
This site gives a more general description of Loop assembly and its features.

uLoop.org
This site is a platform for Universal Loop assembly to enable tracing available DNA parts, linking protocols and other labs using it. It's divided according to model organism.

Recode2S
A web application is a domestication tool to screen for BsaI and SapI sites and add overhangs according to the Common Syntax (Patron et al., 2015) for the uLoop implementation. It also incorporates a domestication mode for the pL0R vectors.

uLoopdesigner
A demo web site for simulating assemblies and keeping a library of assembled parts. Its a fork of the original Loopdesigner with the uLoop implementation of the common sintax.

Plasmapo
A pseudocode plasmid generator with SBOL compatible glyphs.

DNArchive
A experimental community-based DNA database with proper referencing and visualisation.
